構造化のフレームワーク ​

プロンプトを効果的に作成するために、情報を整理し構造化するフレームワークをご紹介します。これらのフレームワークを使うことで、より正確で期待通りの結果を生成AIから得ることができます。

構造化フレームワーク ​

構造化フレームワークの一例を紹介します。

RTFフレームワーク ​

Role（役割）、Task（タスク）、Format（出力形式）の3つの要素でプロンプトを構造化します。

  • Role（役割）：AIにどのような立場や専門性を持ってもらうかを指定
  • Task（タスク）：AIに具体的に何をしてもらうかを明確に指示
  • Format（出力形式）：結果をどのような形式で出力してもらうかを指定

CAREフレームワーク ​

Context（背景）、Action（行動）、Result（結果）、Example（例）の4つの要素で構成します。

  • Context（背景）：タスクを実行する上で必要な背景情報や状況説明
  • Action（行動）：AIに求める具体的な行動や処理内容
  • Result（結果）：出力結果に求める条件や要件
  • Example（例）：期待する出力の具体例を示す

RACEフレームワーク ​

Role（役割）、Action（行動）、Context（背景）、Expectation（期待値）の4つの要素で構成します。

  • Role（役割）：AIに担ってもらう役割や専門分野
  • Action（行動）：実行してもらう具体的な行動
  • Context（背景）：タスク実行に必要な背景情報
  • Expectation（期待する出力の条件）：出力形式や品質に関する期待値

実際にフレームワークを使ってプロンプトを作成してみましょう。

演習1：RTFフレームワーク ​

シナリオ：
新入社員研修で使用する「ビジネスマナー」のチェックリストを作成する必要があります。対象は新卒で入社した20代の社員です。

課題：
RTFフレームワークを使って、ビジネスマナーチェックリスト作成のプロンプトを構成してください。

ヒント：

  • Role: どのような専門性を持った人としてチェックリストを作成してもらいますか？
  • Task: 具体的に何をしてもらいますか？
  • Format: チェックリストはどのような形式で出力してもらいますか？

📝 プロンプト例 ​

# Role（役割）
あなたは人事研修の専門家で、新入社員のビジネスマナー教育に豊富な経験を持っています。

# Task（タスク）
以下の条件で新入社員向けのビジネスマナーチェックリストを作成してください。
- 対象：新卒入社の20代社員（社会人経験なし）
- 目的：日常業務で必要な基本的なビジネスマナーの習得
- 内容分野：挨拶、身だしなみ、電話対応、来客対応、会議マナー
- 使用場面：研修後の自己チェック・上司による評価

# Format（形式）
## ビジネスマナーチェックリスト
### 【分野名】
- [ ] チェック項目（具体的な行動）
- [ ] チェック項目（具体的な行動）

（各分野5項目程度、全25項目程度で構成）

GPT-4oの生成例

新入社員向けビジネスマナーチェックリスト ​

  • [ ] 廊下で上司・先輩とすれ違う時に会釈ができる

【身だしなみ・姿勢】 ​

  • [ ] 清潔感のある服装を心がけている
  • [ ] 髪型が整っており、派手すぎない
  • [ ] 座る時に背筋を伸ばして正しい姿勢を保てる
  • [ ] 歩く時の姿勢が良く、だらしなくない
  • [ ] 名刺交換時に適切な立ち姿勢を取れる

【電話対応】 ​

  • [ ] 電話に出る時「お疲れさまです、○○です」と名乗れる
  • [ ] 相手の会社名・名前を復唱して確認できる
  • [ ] 電話を切る時は相手が切るのを待てる
  • [ ] 聞き取れない時は「恐れ入りますが、もう一度お願いします」と言える
  • [ ] メモを取りながら電話対応ができる

【来客対応】 ​

  • [ ] 来客時に立ち上がって挨拶ができる
  • [ ] 「いらっしゃいませ」「お疲れさまです」を適切に使い分けられる
  • [ ] お客様を会議室まで案内する時、適切な位置を歩ける
  • [ ] お茶出しの際、正しい順番とマナーで提供できる
  • [ ] 来客が帰られる時はエレベーターまで見送りができる

  • [ ] 会議開始5分前には席に着いている
  • [ ] 会議中はスマートフォンをマナーモードにしている
  • [ ] 発言する時は相手の話を最後まで聞いてから話す
  • [ ] 議事録やメモを積極的に取っている
  • [ ] 会議室を使った後は椅子を元の位置に戻し、整理整頓している

演習2：CAREフレームワーク ​

シナリオ：
部下の営業成績が低迷しており、1on1面談でアドバイスをする必要があります。相手は入社3年目の営業担当者で、最近自信を失っているようです。

課題：
CAREフレームワークを使って、1on1面談での会話シナリオ作成のプロンプトを構成してください。

ヒント：

  • Context: どのような背景や状況で面談を行いますか？
  • Action: AIに何を作成してもらいますか？
  • Result: シナリオにはどのような内容や条件が必要ですか？
  • Example: どのような構成や形式の例を示しますか？

📝 プロンプト例 ​

# Context（背景）
入社3年目の営業担当者Aさんは、ここ3か月間目標を達成できず、営業成績が低迷している状況です。最近は表情も暗く、自信を失っているように見えます。管理職として1on1面談を行い、Aさんのモチベーション向上と具体的な改善策を一緒に考える必要があります。

# Action（行動）
建設的で前向きな1on1面談の会話シナリオを作成してください。

# Result（結果）
- 面談時間：30分程度を想定
- Aさんの気持ちに寄り添いながら、問題の本質を探る内容
- 具体的な改善アクションプランを一緒に作成する流れ
- Aさんが前向きな気持ちになれる締めくくり

# Example（例）
会話シナリオの構成例：
1. アイスブレイク・関係構築
2. 現状確認（事実ベース）
3. 課題の整理
4. 改善策の検討
5. アクションプランの作成
6. フォローアップの約束
（各段階での具体的な会話例を示してください）

💻 生成例 ​

GPT-4oの生成例

面談の流れ（30分想定） ​

1. アイスブレイク・関係構築（5分） ​

管理職：「お疲れさまです。今日は貴重な時間をありがとう。最近どう？体調は大丈夫？」

Aさん：「お疲れさまです。体調は大丈夫です...」

管理職：「そうですね。実は今日は、Aさんと一緒に今後のことを考えたいと思って時間を作ったんです。まずは、最近の仕事で何か困っていることはありませんか？」

2. 現状確認（事実ベース）（8分） ​

管理職：「数字の話になってしまいますが、ここ3か月の実績について、Aさん自身はどう感じていますか？」

Aさん：「正直、目標に全然届いていなくて...申し訳ありません」

管理職：「謝る必要はありません。大切なのは現状を正しく理解することです。具体的に、どの段階で苦戦していると感じますか？新規開拓？提案？クロージング？」

Aさん：「提案の段階で断られることが多くて...」

3. 課題の整理（10分） ​

管理職：「提案で断られる理由について、お客様から何かフィードバックをもらったことはありますか？」

Aさん：「『他社の方が安い』とか『タイミングが悪い』とよく言われます」

管理職：「なるほど。それは辛いですね。ただ、そういう時こそ学びのチャンスかもしれません。『他社の方が安い』と言われた時、Aさんはどんな対応をしていますか？」

管理職：「一緒に考えてみましょう。価格で勝負するだけが営業ではないですよね。Aさんの強みって何だと思いますか？」

4. 改善策の検討（5分） ​

管理職：「お客様のニーズをもっと深く聞く時間を作ることはできそうですか？例えば、『予算感を教えてください』の前に『今一番困っていることは何ですか？』って聞いてみるとか。」

Aさん：「そうですね...確かに、もう少しお客様の話を聞く時間を作った方がいいかもしれません」

管理職：「それと、先輩の〇〇さんの商談に同行させてもらうのはどうでしょう？実際の進め方を見るのも勉強になると思います」

5. アクションプランの作成（2分） ​

管理職：「では、来月に向けて具体的に何から始めましょうか？」

Aさん：「お客様の話をもっと聞くことと、先輩の同行をお願いしてみます」

管理職：「素晴らしいですね。それと、私も一緒に大きな案件は同行しますから、遠慮なく声をかけてください」

6. フォローアップの約束（2分） ​

管理職：「2週間後にまた時間を作って、進捗を確認させてもらえますか？きっと良い変化があると思います」

Aさん：「ありがとうございます。頑張ってみます」

管理職：「Aさんには必ず良い営業になる素質があります。一人で抱え込まず、困った時はすぐに相談してくださいね。応援しています！」

演習3：RACEフレームワーク ​

シナリオ：
新商品発売に向けて競合他社の分析レポートを作成する必要があります。マーケティングチームで使用し、今後の戦略立案に活用する予定です。

課題：
RACEフレームワークを使って、競合分析レポート作成のプロンプトを構成してください。

ヒント：

  • Role: どのような立場の人としてレポートを作成してもらいますか？
  • Action: 具体的にどのような分析レポートを作ってもらいますか？
  • Context: 分析を行う背景や制約はありますか？
  • Expectation: レポートの出力にはどのような条件を求めますか？

📝 プロンプト例 ​

# Role（役割）
あなたはマーケティング戦略の専門家で、競合分析と市場調査に精通したビジネスアナリストです。

# Action（行動）
新商品「スマートウォッチ」の競合他社分析レポートを作成してください。

# Context（背景）
- 自社：中堅電機メーカー、スマートウォッチ市場への新規参入
- 予算：ミドルレンジ価格帯（3-5万円）での展開を予定
- ターゲット：健康志向の30-50代ビジネスパーソン
- 発売予定：6か月後
- 用途：マーケティングチーム内での戦略検討会議で使用

# Expectation（期待する出力の条件）
以下の要素を含む分析レポート：
- エグゼクティブサマリー（重要なポイントを3つ）
- 主要競合3社の詳細分析（強み・弱み・価格・特徴）
- 市場のトレンドと機会
- 自社の差別化ポイント提案
- 推奨する次のアクション
- A4で3-4ページ程度のボリューム

💻 生成例 ​

GPT-4oの生成例

**本レポートは、当社が6 か月後に予定するミドルレンジ価格帯（3〜5 万円）の新型スマートウォッチに向け、国内市場と主要競合3社（Apple／Samsung／Garmin）を俯瞰し、差別化ポイントと具体的アクションを整理したものです。IDCの最新データによれば国内ウェアラブル市場は前年比13.5 %増で成長を維持しており、Appleが依然42.8 %のシェアを握りますが、3〜5 万円のミドルレンジ帯が伸長していることが大きな追い風です。([マイIDC][1]) さらに健康意識の高まりと企業のウェルネス投資が需要を押し上げており、当社が狙う30〜50代ビジネスパーソンの“健康×生産性”ニーズが顕在化しています。([M2RI][2], [Prone][3])

エグゼクティブサマリー ​

  1. 市場は右肩上がり、ミドルレンジ帯が空白地帯
    2024年の国内スマートウォッチ出荷は対前年比13.5 %増。ハイエンドとローエンドの品揃えは飽和しつつあり、3〜5 万円帯に伸びしろがある。([マイIDC][1])
  2. 競合はエコシステム or フィットネス特化で二極化
    Apple／SamsungはOS連携によるユーザー囲い込みが強み。一方Garminはロングバッテリーと高度フィットネス指標で存在感。価格・機能のバランスにギャップ。([Tom's Guide][4], [Android Authority][5], [The Verge][6])
  3. 企業向け健康経営需要が拡大
    ウェアラブルによる従業員ヘルスモニタリング事例が増加し、福利厚生の一環として採用が進む。BtoB2Cモデルは当社の新規参入機会。([Prone][3], [プレスリリース・ニュースリリース配信シェアNo.1｜PR TIMES][7])

競合間のポジショニング ​

  • Apple：エコシステム＆決済で囲い込み。短寿命バッテリーがビジネスユーザー不満点。
  • Samsung：Android陣営旗艦。AI音声で先行も機能ロックが障壁。
  • Garmin：“ライフログ＋スポーツ”に特化。長時間駆動とヘルスデータの深さが武器。

市場トレンドと機会 ​

1. 健康×AIの高度化 ​

ストレス・睡眠・心臓年齢などソフト推定指標が洗練され、AIがコーチングまで実装される流れ。([istarmax.com][16], [Tom's Guide][12])

2. 医療グレード機能の規制緩和 ​

心電図（ECG）機能は2020年以降Apple等が先行して国内承認、他社も追随。([Smart Watcher][17])

3. バッテリー／耐用年数への関心 ​

ユーザーが寿命3年を意識し、長持ちバッテリーと交換サービスが評価軸。([CCガジェット][18])

4. 企業ウェルネスと保険連動 ​

FitbitやGarminを使った活動量インセンティブ制度が拡大。([Prone][3], [プレスリリース・ニュースリリース配信シェアNo.1｜PR TIMES][7])

5. 決済・交通ICの拡張 ​

Suica完全対応はAppleが先行し、Garmin/Fitbitは機能限定。乗車・定期・オートチャージフル対応は大きな差別化余地。([PC Watch][11])

自社の差別化ポイント提案 ​

  1. “5 days+”バッテリー & 60 min急速充電
    ビジネス出張でも充電ストレスを排除し、Appleとの差別化。
  2. 法人ウェルネスダッシュボード
    企業向けに従業員の活動データを匿名集計し、健康経営KPIを可視化。
  3. Suicaフル機能＋マルチペイ
    定期券・オートチャージ対応で通勤効率を向上。
  4. AIストレスマネジメント
    心拍変動・皮膚温度をAI解析し、会議前リラックス提案や“集中モード”切替。
  5. データプライバシー・日本語UI最適化
    国内サーバー保管＆契約書テンプレートでB2B導入ハードルを低減。

まとめ
ミドルレンジ帯を狙う当社は、(1)長時間バッテリー、(2)法人ウェルネス連携、(3)Suicaフル対応という3点を核に、エコシステム重視のApple／Samsungとフィットネス特化のGarminの間にある“ヘルス＆ビジネス”ニッチを占有できます。上記アクションを迅速に実行し、6 か月後のローンチに向け競合優位を確立してください。
